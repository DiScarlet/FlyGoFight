Fly Go Crash
Overview

This is a 2D physics-based game where players launch birds using a slingshot to destroy enemy structures and gain kebabs to feed its viallage. 

Gameplay:
1. Drag the bird from the slingshot to aim.
2. Release the bird to launch.
3. Hit monsters, knok down structures and gain kebabs to complete the level.
4. Each level contains different types of obstacles so carefully read the hints.

Project structure:
Assets/
 ├─ Additional/      # Additional elements of design
 ├─ Animations/      # Transition animations
 ├─ Birds/           # Sprites of birds
 ├─ fonts/           # Fonts used in a game
 ├─ Materials/       # Materials
 ├─ Birds/           # Sprites of monsters
 ├─ Prefabs/         # Prefabs of birds, enemies, slingshot, etc
 ├─ Scenes/          # Scenes for menus 
    ├─ Levels/       # Scenes of levels
 ├─ Script/          # C# scripts for gameplay, camera, and UI
 ├─ Slingshot/       # Slinshot elements
 ├─ Targets/         # Crates
 ├─ TextMesh Pro/    # TextMesh Pro elements
 ├─ Textures/        # Background, kebabs, buttons  
 └─ Transition/      # Transition prefab
